**********************************************************************************************************************************
Command fomat
•     BC <msg>
•     BC-FILE <file>
•     UC <msg> <client_*>
•     UC-FILE <file> <client_*>
•     BLC <msg> <client_*>
•     BLC-FILE <msg> <client_*>
*     BC -> broadcast, UC-> unicast, BLC -> blockcast
